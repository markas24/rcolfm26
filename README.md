# AfricaSoftt DEV Mobile App

Application mobile officielle d'AfricaSoftt DEV - Solutions de développement Web, Android et USSD.

## 📱 À propos

Cette application présente les services d'AfricaSoftt DEV et permet aux clients de contacter facilement l'équipe via WhatsApp et email.

## 🚀 Technologies

- React Native
- Expo
- Expo Linear Gradient
- Expo Vector Icons

## 📦 Installation

```bash
# Cloner le projet
git clone https://github.com/votre-username/africasoftt-dev-mobile.git

# Installer les dépendances
cd africasoftt-dev-mobile
npm install

# Lancer l'application
npm start
# ou
expo start
